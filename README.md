# 🤟 Arabic Sign Language Recognition (ArabicSL)

## 📌 Overview
This competition focuses on recognizing Arabic sign language alphabets.

## 🎯 Task Description
Classify images into 32 Arabic alphabet signs.

## 🏆 Objective
Achieve high accuracy in sign recognition.

## 🔗 Dataset Link
https://www.kaggle.com/datasets/gannayasser/arabic-alphabets-sign-language-dataset-arasl

## 📂 Dataset Description
Dataset contains labeled hand gesture images for Arabic alphabets.

## 📁 Dataset Structure
train/, test/

### Public vs Private
train public, test labels private

### 📊 Evaluation Metrics
To ensure the robustness of the models, we evaluate participants based on:
*   **Accuracy:** Overall percentage of correct leaf classifications.
*   **F1-Score:** The harmonic mean of precision and recall (Macro-averaged), which is our primary ranking metric.

You can view the full statistical breakdown of all teams here: [Interactive Results Table](https://tasneem-mselim.github.io/PLT/)
